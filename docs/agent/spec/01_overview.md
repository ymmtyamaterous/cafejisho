# Cafe Jisho — 全体概要仕様書

> 作成日: 2026-03-28

---

## 1. プロダクト概要

| 項目 | 内容 |
|------|------|
| プロダクト名 | Cafe Jisho（カフェ辞書） |
| コンセプト | コーヒーについてゲーム感覚で楽しく学べる学習プラットフォーム |
| ターゲット | コーヒー初心者〜中上級者、コーヒーに興味を持つ幅広いユーザー |
| リリース形態 | Web アプリ（レスポンシブ対応） |
| バージョン | β |

---

## 2. 技術スタック

| レイヤー | 技術 |
|----------|------|
| フロントエンド | TypeScript / React / TanStack Router / Vite |
| バックエンド | TypeScript / Hono / Bun |
| API 通信 | oRPC（型安全な RPC） |
| 認証 | better-auth |
| ORM | Drizzle ORM |
| データベース | SQLite（libsql） |
| モノレポ管理 | Turborepo |
| リンター / フォーマッター | Biome |
| Git フック | Lefthook |
| UI コンポーネント | shadcn/ui (Tailwind CSS ベース) |

### パッケージ構成

```
better-t-app/
├── apps/
│   ├── web/         # フロントエンド (React + TanStack Router)
│   └── server/      # バックエンド (Hono + Bun)
└── packages/
    ├── api/         # oRPC ルーター定義
    ├── auth/        # better-auth 設定
    ├── db/          # Drizzle スキーマ + クライアント
    ├── env/         # 環境変数バリデーション
    ├── ui/          # 共通 UI コンポーネント
    └── config/      # 共通 TypeScript 設定
```

| ロール | 説明 | アクセス可能コンテンツ |
|--------|------|----------------------|

## 5. コンテンツ構成

### 5.1 コース

| コース | レベル | レッスン数 | 所要時間 | 料金 |
|--------|--------|-----------|---------|------|
| はじめてのコーヒー学 | 入門 (BEGINNER) | 8 | 約2時間 | 無料 |
| 産地と焙煎の科学 | 中級 (INTERMEDIATE) | 14 | 約5時間 | プレミアム |
| カッピングとスペシャルティ | 上級 (ADVANCED) | 20 | 約8時間 | プレミアム |

### 5.2 辞典

- コーヒー関連用語を120件以上収録
- カテゴリ別（豆・焙煎・抽出・産地・テイスティング 等）に分類
- 用語の読み仮名・英語名・説明文を掲載

### 5.3 産地ガイド

- 世界30産地・農園の詳細情報
- 産地ごとの特徴フレーバー・標高・生産量・品種を掲載

### 5.4 クイズ

- 各コース / レッスンに紐づくクイズ
- 4択選択式
- 正解率・スコアの記録

## 8. デザイン方針

- **カラーパレット**: コーヒーをイメージした暖色系（エスプレッソ / ラテ / クリーム）
- **フォント**: Nunito（本文）、Zen Maru Gothic（見出し）
- **スタイル**: ネオブルータリスト（太いボーダー・ハードシャドウ）
- **インタラクション**: ホバー時の微細な移動とシャドウ変化でリッチな体験

| 変数名 | カラーコード | 用途 |
|--------|------------|------|
| `--espresso` | `#2C1A0E` | 主要テキスト・ボーダー・背景 |
| `--roast` | `#6B3D1E` | サブテキスト・アクセント |
| `--latte` | `#C49A6C` | 中間アクセント |
| `--latte-light` | `#E8C99A` | ハイライト・バッジ背景 |
| `--cream` | `#F5EFE0` | カードやボタン背景 |
| `--bg` | `#FAF7F2` | ページ背景 |
