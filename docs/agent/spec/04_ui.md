# Cafe Jisho — UI / 画面設計書

> 作成日: 2026-03-28

---

## 1. デザインシステム

### 1.1 カラーパレット

| CSS 変数 | 値 | 用途 |
|----------|----|------|
| `--espresso` | `#2C1A0E` | 主要テキスト・ボーダー・ダーク背景 |
| `--roast` | `#6B3D1E` | サブテキスト・アクセント |
| `--latte` | `#C49A6C` | 中間アクセント・バッジ |
| `--latte-light` | `#E8C99A` | ハイライト・ピル背景 |
| `--cream` | `#F5EFE0` | カード・ボタン背景 |
| `--steam` | `#EDE5D4` | セカンダリ背景 |
| `--bg` | `#FAF7F2` | ページ背景 |
| `--white` | `#FFFFFF` | 純白背景 |

### 1.2 タイポグラフィ

| 用途 | フォント | ウェイト |
|------|---------|---------|
| 見出し（H1〜H3） | Zen Maru Gothic | 900 |
| 本文・UI テキスト | Nunito | 400 / 700 / 800 |
| バッジ・ラベル | Nunito | 900 |

### 1.3 コンポーネントスタイル（ネオブルータリスト）

- **ボーダー**: `2px〜2.5px solid var(--espresso)`
- **ボーダー半径**: `10px〜24px`（要素サイズに応じて）
- **ハードシャドウ**: `4px 4px 0 var(--espresso)` ※深みに応じて変化
- **ホバー効果**: `transform: translate(-2px, -2px)` + シャドウ拡大
- **トランジション**: `0.1s〜0.15s` でスピーディに

### 1.4 ドットパターン背景

- ページ背景に薄いドットグリッドを全面に敷く（`::before` 疑似要素）
- `radial-gradient(circle, rgba(196,154,108,0.2) 2px, transparent 2px)`
- `background-size: 26px 26px`

## 3. 共通レイアウト

### 3.1 ナビゲーションバー

- 固定（`position: sticky top-0`）、ページ上部に常時表示
- 背景: `var(--white)` + ボーダー + ハードシャドウ (`5px 5px 0 var(--espresso)`)
- ボーダー半径: `24px` (フローティングスタイル)
- 内容:
  - 左: ロゴ「☕ cafejisho」+ β バッジ
  - 中央: ナビリンク（コース / 辞典 / 産地 / クイズ）
  - 右: 未ログイン時「はじめる ✨」ボタン / ログイン時ユーザーメニュー

### 3.2 フッター

- 背景: `var(--espresso)`、ボーダー上部: `3px solid var(--latte)`
- 4カラムグリッドレイアウト
  - 左: ロゴ・キャッチコピー・SNSリンク（X / Instagram / YouTube）
  - 学ぶ: 入門コース / 産地辞典 / 焙煎ガイド / 抽出法一覧
  - 辞典: 用語集 / 農園マップ / 品種図鑑 / フレーバー表
  - その他: コミュニティ / マガジン / お問い合わせ / プレミアム ⭐
- 最下部: コピーライト・プライバシー/利用規約リンク

## 4. 各画面の詳細

### 4.1 ランディングページ (`/`)

#### セクション構成

```
[ナビゲーションバー]
[ヒーローセクション]
  - 左: キャッチコピー + CTA ボタン
  - 右: コンテンツカルーセル
[統計セクション]
[ウェーブ区切り]
[コース紹介セクション]
[ウェーブ区切り（反転）]
[フッター]
```

#### ヒーローセクション

- **レイアウト**: 2カラムグリッド（左: テキスト / 右: カルーセル）
- **左カラム**:
  - ピル型バッジ「☕ コーヒーを楽しく学ぼう！」
  - H1「コーヒーってこんなにおもしろい！」（マーカーハイライト付き）
  - サブテキスト
  - CTA ボタン 2 本:「無料で学ぶ 🚀」（プライマリ）・「クイズに挑戦 🎯」（セカンダリ）
- **右カラム（カルーセル）**:
  - 4 スライド（今日のレッスン / 産地ピックアップ / 焙煎の科学 / 今日のクイズ）
  - 前後矢印ボタン + ドットインジケーター + 4秒自動スライド
  - 各スライド: タグ・絵文字・タイトル・説明・「学ぶ →」ボタン

#### 統計セクション

- 4カラムグリッド（カード型）
  - `120+` ☕ コーヒー用語
  - `30` 🌍 産地・農園
  - `12` 🔬 抽出法ガイド
  - `無料` 🎉 基礎コース全部
- 各カードにホバーアニメーション

#### コース紹介セクション

- 背景: `var(--espresso)` のダークセクション
- 3カラムグリッドでコースカード表示
  - 入門: 🌱・8レッスン・2時間・無料
  - 中級: 🫘・14レッスン・5時間
  - 上級: 🏆・20レッスン・8時間

---

### 4.2 ログイン画面 (`/login`)

- 中央カード型フォーム
- メールアドレス・パスワード入力フィールド
- 「ログイン」ボタン（プライマリ）
- 「アカウントをお持ちでない方はこちら」→ `/signup` リンク
- バリデーションエラーをインラインで表示

---

### 4.3 サインアップ画面 (`/signup`)

- 中央カード型フォーム
- 名前・メールアドレス・パスワード・パスワード確認入力フィールド
- 「アカウント作成」ボタン（プライマリ）
- 「すでにアカウントをお持ちの方はこちら」→ `/login` リンク

---

### 4.4 ダッシュボード (`/dashboard`)

#### レイアウト

```
[ウェルカムメッセージ]
[学習統計バー（完了レッスン数 / 学習継続日数 / 平均クイズスコア）]
[コース進捗セクション]
  - 各コースの進捗バー
[最近の学習セクション]
  - 最近完了したレッスンカード（最大5件）
[おすすめ次のレッスン]
```

#### 統計バー

- 水平スクロール可能なカード群
- アイコン + 数値 + ラベル

---

### 4.5 コース一覧 (`/courses`)

- フィルターボタン（全て / 入門 / 中級 / 上級）
- カードグリッド（PC: 3列 / タブレット: 2列 / SP: 1列）
- 各カード:
  - 絵文字・レベルバッジ・タイトル・説明・レッスン数と所要時間・プレミアムバッジ（該当時）
  - 進捗バー（ログイン時）

---

### 4.6 コース詳細 (`/courses/:courseId`)

- ヒーローバナー（コース名・説明・統計）
- レッスン一覧（アコーディオンまたはリスト）
  - 各行: 番号・タイトル・所要時間・完了チェックマーク（ログイン時）
- 「コースを始める」/ 「続きから」CTA ボタン
- プレミアムコースの場合: ロックオーバーレイ + 「プレミアムで解放」ボタン

---

### 4.7 レッスン詳細 (`/courses/:courseId/lessons/:lessonId`)

- 上部: パンくずリスト（コース → レッスン名）
- サイドバー（PC のみ）: レッスン一覧・進捗インジケーター
- メインコンテンツ: Markdown レンダリングされた本文
  - コードブロック・画像・見出しにコーヒースタイルを適用
- 下部:
  - 前後レッスンへのナビゲーション
  - 「完了する」ボタン（ログイン時）→ 完了するとクイズへ遷移
  - ブックマークボタン

---

### 4.8 クイズ (`/quiz/:quizId`)

#### 回答中

- プログレスバー（問題 N / 全問数）
- 問題文
- 4択ラジオボタン（カード型、選択時にハイライト）
- 「次へ」ボタン

#### 結果画面

- スコア表示（大きな円グラフ or 数字）
- 全問の正誤リスト + 解説テキスト
- 「もう一度挑戦」・「次のレッスンへ」ボタン

---

### 4.9 辞典 (`/glossary`)

- 検索バー（リアルタイムフィルター）
- カテゴリタブ（全て / 豆・品種 / 焙煎 / 抽出 / 産地 / テイスティング / 器具 / 認証）
- 用語カードグリッド
  - 用語名・読み仮名・英語名・短い説明

---

### 4.10 用語詳細 (`/glossary/:termId`)

- 大見出し（用語名 + 読み仮名 + 英語名）
- カテゴリバッジ
- 詳細説明（Markdown レンダリング）
- 関連用語カード（横スクロール）
- ブックマークボタン

---

### 4.11 産地一覧 (`/origins`)

- 大陸フィルタータブ（全て / アフリカ / アジア / 中南米 / その他）
- 産地カードグリッド（PC: 4列 / タブレット: 2列 / SP: 1列）
  - 国旗 / 絵文字・産地名・フレーバータグ・標高

---

### 4.12 産地詳細 (`/origins/:originId`)

- ヒーローバナー（産地名・大陸・国）
- フレーバータグ一覧
- 詳細情報グリッド（標高 / 年間生産量 / 主要品種 / 精製方法）
- 説明文（Markdown レンダリング）
- ブックマークボタン

---

### 4.13 プロフィール (`/profile`)

- アバター画像 + 名前・メール・登録日
- 編集フォーム（名前・アバター）
- 統計サマリー（完了レッスン数 / クイズ平均スコア）
- 取得バッジ一覧（将来拡張）

---

### 4.14 ブックマーク (`/bookmarks`)

- タブ切り替え（レッスン / 用語 / 産地）
- 各タブのカードリスト
- ブックマーク削除ボタン

---

### 4.15 プレミアムプラン (`/premium`)

- プラン比較テーブル（無料 vs プレミアム）
- 「プレミアムに登録する」CTA ボタン
- よくある質問（FAQ）セクション

## 6. インタラクション一覧

| UI 要素 | インタラクション |
|---------|----------------|
| ボタン（プライマリ / セカンダリ） | ホバー: `-2px,-2px` 移動 + シャドウ拡大 |
| カード全般 | ホバー: `-2px,-2px` 移動 + シャドウ拡大 |
| ナビリンク | ホバー: `-1px,-1px` 移動 |
| カルーセル | スワイプ / 矢印 / ドット / 4秒自動送り |
| クイズ選択肢 | 選択時: `--espresso` 背景に切替 + チェックアイコン |
| 完了ボタン | クリック後: チェックアイコン + 緑色トーストでフィードバック |
| ブックマーク | トグルでアイコン切替（塗りつぶし / アウトライン） |
| 進捗バー | アニメーション付きで塗りつぶし |
